# Codex 技能管理器 v0.1.2

本版本重点优化在线 Skills 广场：打开更快、来源切换更清楚、扫描动作更可控。

## 主要变化

- 打开技能广场时只读取本地缓存，不再自动联网扫描所有来源。
- 点击左侧来源卡片只切换筛选，不会误触发慢速 GitHub 深度扫描。
- 新增“扫描来源 / 刷新来源”独立动作，只扫描用户明确选择的来源。
- 来源卡片会显示状态：未扫描、可浏览、未发现、扫描失败、外部目录。
- 技能广场缓存改为按来源保存，避免不同来源搜索结果串台。
- 外部链接统一交给系统浏览器打开。

## 修复

- 修复点击空索引来源时应用像卡住一样等待的问题。
- 修复 GitHub Topic 或外部目录来源被误当成可深度扫描仓库的问题。
- 修复 marketplace 搜索缓存未包含来源维度的潜在边界问题。

## 验证

- `npm run test:run`
- `npm run build`
- 敏感信息扫描
